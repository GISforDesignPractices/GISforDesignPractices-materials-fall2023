In-class Exercises

Data Plotly

  1. Download today's data.
  2. Open QGIS and open the shapefile in ga-affordability.
  3. Install the Data Plotly plugin.
  4. Open Data Plotly: Plugins > Data Plotly > Data Plotly plugin.
  5. Make a scatterplot of mpv_n (median property value) versus aff_ratio (affordability ratio).
  6. Select additional hover label = city.
  7. Explore outliers by clicking on the dots in the scatterplot and finding them in your map.
  8. In the Layout Options, set X Axis Bounds and/or Y Axis Bounds so that they exclude outliers.
