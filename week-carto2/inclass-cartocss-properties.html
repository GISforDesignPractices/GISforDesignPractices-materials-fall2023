In-class Exercises

Changing properties with CartoCSS

  1. Continue with the Airbnb data from last time we talked about Carto. If you need it, upload the nyc-listings-2022-09.geojson file from today's data to Carto.
  2. Open the CartoCSS documentation.
  3. Make a map with the data and style the points.
  4. Using CartoCSS:
    1. Make the markers 5 pixels wide.
    2. Give the markers an outline 2 pixels wide.
    3. Give the fill and stroke of the markers different colors.
    (The exact values are less important than figuring out how to set the size of the markers with CartoCSS, so feel free to experiment with values other than 5 and 2.)
  5. Keep this Carto map open for upcoming exercises.
