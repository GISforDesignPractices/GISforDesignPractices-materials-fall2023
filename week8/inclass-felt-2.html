In-class Exercises

Data clean-up + Felt

  1. Download the latest data from Inside Airbnb for NYC. You can view all available data on their site.
  2. Unzip the data.
  3. Open the data in Excel or another spreadsheet application.
  4. Try to get the file size down to around 70 MB by deleting some of the larger fields we do not need. You'll need to make some judgement calls here but we definitely want:
    • availability_365
    • id
    • latitude
    • listing_url
    • longitude
    • price
    • room_type
  5. Delete columns, save the file, and check the new size. Columns with a lot of text are the best ones to go for to try to get data smaller.
  6. Make the price column numeric by using Find and Replace to remove:
    1. $
    2. ,
  7. Log in to your Felt account.
  8. Create a new map.
  9. Upload the modified CSV file (this will take a minute or two as the file is still likely large).
  10. Change the styles of the points:
    1. Style the points by value with the room_type column to make a categorized style.
    2. Style the points by value with the price column to make a graduated style.
  11. Experiment with some of the features we just talked about:
    • changing background styles
    • using the library to add more data to the map
    • using routes and other annotations
  12. Get the public link for the map.
