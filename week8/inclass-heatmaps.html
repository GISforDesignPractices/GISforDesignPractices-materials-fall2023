In-class Exercises

Heatmaps

Using the Low Income Housing Tax Credit (LIHTC) Development points layer, add a heatmap of low-income rental properties created through the LIHTC program city of your choice.

  1. If you haven't already, download today's data.
  2. Add the Low Income Housing Tax Credit (LIHTC) csv to QGIS and map the points using longitude (x field) and latitude (y field).
  3. Use the Field Calculator to create a new field that fixes li_units to be numeric. Make the type integer and use the expression if("li_units" = 'NA', null, to_int("li_units")). That is, if li_units has "NA" in it set the value to null (empty), else convert it to an integer.
  4. Toggle editing off to save your changes.
  5. Add the Borough Boundaries shapefile.
  6. Use Select by Location to select the LIHTC buildings in New York City.
  7. Export the LIHTC Listings data with 2263 as the CRS.
  8. Search for Heatmap (Kernel Density Estimation) in the QGIS search box.
  9. Weight the heat map based on the number of affordable units in the building. Set Weight from field to your numeric li_units field.
  10. Set the radius to 2000 feet.
  11. Set the rows to 3000. This is the number of rows of pixels in the resulting raster. The columns should update automatically.
  12. Style the output raster with a Singleband Pseudocolor render type and select a color ramp.
  13. Create a few more heatmaps with varying radii and resolution (rows and columns).
