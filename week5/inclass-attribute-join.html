Attribute Joins

  1. If you haven't already, download today's data.
  2. Open the cb_2018_us_state_500k shapefile in QGIS.
  3. Open the Excel file in nytimes-covid-us-states named us-states-20221009.xlsx in QGIS. You can drag and drop it into QGIS.
  4. Attribute join the COVID data to the states. Use fips as the Join field, STATEFP as the Target field.
  5. We'll come back to this dataset for the next in-class exercise.
