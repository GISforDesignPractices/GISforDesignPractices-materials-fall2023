Add your own data to a Mapbox style

  1. Mapbox Basic also doesn't contain the borough boundaries. Let's add those now.
  2. Download today's data and unzip it. The zip file nybb.zip contains the borough boundaries shapefile.
  3. Add a new tileset in your Mapbox account and upload nybb.zip.
  4. Back in your style, add a new layer and select the borough boundaries tileset.
  5. Style the borough boundaries.
