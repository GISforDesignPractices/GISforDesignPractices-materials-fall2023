In-class Exercises

Mapbox

  1. Create and log in to a Mapbox account and go to Studio.
  2. Create a new style based on Streets.
  3. Change the color of some polygons and lines.
  4. Change the font of some labels.
  5. Try making some zoom-dependent changes such as making buildings more transparent when zoomed out and less transparent when zoomed in.
  6. Experiment!
