Temporal Controller

  1. Download today's data if you haven't already.
  2. Open the shapefile in the Blue whales Eastern North Pacific 1993-2008 - Argos data folder named points-fixed.shp in QGIS. This is data via this movebank study looking at blue whale migrations.
  3. Open the Temporal Controller panel in QGIS (View > Panels > Temporal Controller).
  4. Open the points-fixed layer properties (right-click, Properties).
  5. Go to the Temporal tab.
  6. Check the box next to Temporal to enable it, then select Single Field with Date/Time from the Configuration dropdown. Set the Field to date.
  7. Click Ok to close the properties window.
  8. Change some of the settings in the Temporal Controller panel to make an animation using the whale location data. You may need to change the frame rate (in the settings) and the step length to get a smooth animation.
