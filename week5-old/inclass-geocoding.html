Geocoding Example

  1. We will geocode an exerpt of the 40,329 applications for a license to sell tobacco in New York City since 1998.
  2. First you need to make sure there is an address column with building number and street in one field.
  3. In Excel, open license_applications/tobacco_excerpt.csv and create a street address field from two columns: "Building Number" and "Street". You can use the CONCATENATE function.
  4. Open QGIS and install the MMQGIS Plugin.
  5. In the new MMQIS dropdown, select Geocode > Geocode CSV with Web Service.
  6. In this dialog:
    1. Open the version of tobacco_excerpt.csv you created with an address field.
    2. Set Web Service = US Census Bureau
    3. Name your Output File Name and Not Found Output List.
    4. Click Apply
  7. Add a baselayer using QuickMap Web Service to see where the geocoded points are
  8. Open the Not Found Output List to see if any stores were not successfully geocoded.
  9. If you can determine the error in the address of the ungeocoded addresses, you may want to correct the addresses and geocode again.
