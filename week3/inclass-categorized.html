In-class Exercises

Categorized maps

  1. Open the streets shapefile in QGIS.
  2. Style using a Categorized style on the column RW_TYPE. Customize some categories' styles and label text using the metadata for the file (in the same folder as the shapefile).
  3. Open the NYPD Motor Vehicle Collisions CSV in QGIS if it's not already open.
  4. Style using a Categorized style on the column CONTRIBUTING FACTOR VEHICLE 1.
  5. Try categorizing on some other fields.
