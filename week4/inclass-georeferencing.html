In-class Exercises

Georeferencing

  1. We will do this exercise together.
  2. Find an image to georeference:
    1. Go to Urban Reviewer.
    2. Find an urban renewal plan with an image, such as Brooklyn Navy Yards.
    3. Click on the image to make it larger, then right click on it and save as to save it to your computer.
  3. Prepare QGIS:
    1. Open QGIS.
    2. Install the QuickMapServices plugin if you don't have it yet.
    3. Set the project's projection (CRS) to EPSG:3857. We do this because the base layer in the next step is in that projection.
    4. Using QuickMapServices, add a satellite base layer with labels. I would suggest the Google Hybrid layer.
    5. Zoom to the area your urban renewal plan should be in.
  4. Go to Raster > Georeferencer > Georeferencer to open the Georeferencer window.
  5. Go to File > Open Raster and select the image you downloaded from Urban Reviewer.
  6. Wait here and I'll walk you through the next step.
  7. Add 3 or 4 ground control points:
    1. Switch between the Georeferencer and map windows and find a common point between the two (generally a street corner will be best).
    2. Click the Add point button in the Georeferencer window.
    3. Click on the common point in the Georeferencer window.
    4. When prompted to Enter map coordinates, select the From map canvas button.
    5. Click on the common point on the map.
  8. Once you are happy with the ground control points, click the Settings icon in the Georeferencer window.
    1. Set Transformation type to Thin plate spline.
    2. Set Target SRS to 3857 (the projection the reference map is in).
    3. Select a location where you want to save the georeferenced image.
    4. Check Load in QGIS when done.
  9. Click Start Georeferencing in the Georeferencer window (looks like a play button). The Georeferenced image should show up on your map.
  10. Refine your georeferenced image:
    1. Find a ground control point that doesn't line up as much as you would like.
    2. Delete the ground control point in the Georeferencer window.
    3. Add a replacement ground control point.
    4. Georeference it again—by default your georeferenced image will be overwritten, so change the destination in the georeferencer settings if you want to make a new file.
