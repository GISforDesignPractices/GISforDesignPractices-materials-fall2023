In-class Exercises

Select by Location

  1. Download this class's data.
  2. Open the Borough Boundaries shapefile in QGIS.
  3. Open the Public_School_Locations shapefile in QGIS.
  4. Use select by expression to select school features in Queens.
  5. Deselect the features.
  6. Use select features by single click (tool) to select the Bronx.
  7. Use select by location to select school features in the Bronx.
  8. Export the selected school features to their own shapefile.
