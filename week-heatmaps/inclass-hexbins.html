Hexbins

Use hexbins to look at the density of points.

  1. Download today's data.
  2. In QGIS, open the file in Inside Airbnb.
  3. Use the QGIS search bar to search for Create grid.
  4. Change Grid CRS to something not in degrees (eg, 2263 is in feet).
  5. Select Hexagon for the grid type, set the grid extent to Calculate from layer > nyc-listings-2022-09.
  6. Set the Horizontal spacing and Vertical spacing to 2500 feet to start.
  7. Create the grid.
  8. If the grid's hexagons are too large or too small, repeat the process but change the Horizontal spacing or Vertical spacing. These determine the height and width of the hexagons.
  9. Count the number of points in each hexagon and make a choropleth with the resulting layer.
